Цвета Указаны в RBGA

Основная карта
1. Прозрачный(0, 0, 0, 0) - Фон
2. Чёрный(0, 0, 0, 1) - Путь
3. Зелёный(0, 1, 0 , 1) - Начало уровня
4. Красный(1, 0, 0, 1) - Конец уровня
5. Серый(0.5, 0.5, 0.5, 1) - Путь с тупиком(нужно указывать два блока рядом)

Дополнительная карта
1. Серый(0.5, 0.5, 0.5, 1) - Чёрные точки
2. Чёрный(n, n, n, 1) - Цветной квадрат
3. Полупрозрачный цвет(n, n, n, 0.5) - Цветные звёздочки
